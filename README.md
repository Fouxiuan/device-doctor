# 设备诊疗师

设备异常闭环与知识管家 - 基于 Web 的轻量级设备异常处理工作台

一个基于 Web 的轻量级设备异常处理工作台，将「状态看板」与「智能知识库」深度绑定，让异常处理从「人追着问题跑」变成「数据推着人走」。

## 功能特性

- **可视化 Kanban 看板** — 5 列覆盖处理全流程（数据校对 → 检测中 → 判定正常/危险 → 穿戴测试），卡片支持拖拽流转和键盘方向键移动
- **结构化智能知识库** — 分类目录 + 标签筛选 + 关键词秒级检索，Markdown 实时双栏预览编辑
- **AI 智能诊疗** — 一键调用大模型，基于异常信息自动推荐标签、阶段、可能原因和处理建议
- **完整数据闭环** — 发现异常 → 指派责任人 → 追踪状态 → 沉淀方案 → 再次检索

## 技术栈

- HTML5 + CSS3 + 原生 JavaScript (ES6+)
- LocalStorage 本地持久化 + JSON 导入导出
- 原生 Drag and Drop API
- 豆包大模型（火山引擎方舟平台）

## 使用方法

用浏览器直接打开 `index.html` 即可体验全部功能。

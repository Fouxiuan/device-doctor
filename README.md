# 设备诊疗师 — 异常闭环与知识管家

> 基于 Web 的轻量级设备异常处理工作台，将状态看板与智能知识库深度绑定

## ✨ 功能特性

### 📊 状态看板
- 五列 Kanban 视图：数据校对 → 检测中 → 判定正常 / 判定危险 → 穿戴测试
- 拖拽卡片即可变更阶段，键盘方向键同样支持
- 实时统计：异常总数、设备总数、解决率、平均耗时
- 点击统计卡片快速筛选对应阶段

### 📱 设备管理
- 完整的设备资产管理（编号、名称、型号、类型、出厂时间、状态、备注）
- 搜索：支持按编号、名称、型号、类型、备注模糊搜索
- 设备状态：使用中 / 维修中 / 已退役
- 自动统计每台设备关联的异常记录数

### 📚 智能知识库
- Markdown 编辑器，支持实时预览
- 文档分类管理，支持展开/折叠
- 标签系统，多标签筛选
- 全文搜索（标题、标签、内容、关联设备）
- 知识库与异常双向关联，异常详情一键跳转对应方案

### 🤖 AI 智能诊疗
- 基于豆包大模型的异常智能诊断
- 自动推荐标签、阶段、可能原因、处理建议
- 点击即可一键应用推荐结果
- API Key 仅存储在本地浏览器，安全可靠

### 💾 数据管理
- 数据存储在浏览器 LocalStorage，离线可用
- 一键导出 JSON 备份
- 支持 JSON 数据导入与合并
- 内置演示数据，首次访问即可体验

## 🚀 快速开始

### 直接打开

直接用浏览器打开 `index.html` 即可使用。

### 本地服务（推荐）

```bash
# 方式一：Python
python3 -m http.server 8080

# 方式二：Node.js
npx serve .
```

然后访问 `http://localhost:8080`。

## 🛠 技术栈

- **纯前端**：HTML + CSS + 原生 JavaScript，无构建依赖
- **UI 设计**：现代化卡片式设计，支持深浅色适配（CSS 变量）
- **响应式**：适配桌面端、平板、手机
- **无障碍**：键盘导航、ARIA 标签、焦点管理
- **动画**：丝滑过渡动画，尊重 `prefers-reduced-motion`

## 🔧 AI 配置

1. 点击右上角 ⚙️ 图标打开「AI 诊疗设置」
2. 输入你的豆包 API Key（获取地址：火山引擎方舟平台）
3. 保存后即可在新增/编辑异常时使用「AI 智能诊疗」功能

> API Key 仅保存在当前浏览器的 LocalStorage 中，不会上传到任何服务器。

## 📝 数据说明

- 所有数据保存在浏览器 LocalStorage
- 清除浏览器数据会导致数据丢失，请定期导出备份
- 首次访问会自动加载演示数据
